# Pepe Match-3

A simple **match-3** puzzle game with crypto / meme coin vibes, built with Python and Pygame.

Swap colorful blocks (inspired by popular tokens like $PEPE, $SHIB, $DOGE, etc.) to create matches of 3 or more.  
Score points, watch them fall, and pump your score to the moon! 🚀🐸

## Features
- 8×8 grid
- 6 different colorful tokens
- Smooth falling animation after matches
- Score counter
- Click-to-select & swap mechanic
- Dark crypto-style theme

## How to Play
1. Click on a tile to select it  
2. Click on an adjacent tile (up/down/left/right) to swap  
3. Create horizontal or vertical matches of **3 or more** same tiles  
4. Matched tiles disappear → new ones fall from the top  
5. Keep playing and rack up points!
